# Expedia-API

Retrieve information from EXPEDIA API to view hotel information

## Start & Install
```
git clone https://github.com/mohamekasem/Expedia-API.git


install Node Package Manager  : npm install

Run the code : npm Start

```


## Built With
* NodeJs
* Express Framework 
* Angular.Js
* Expedia Api
